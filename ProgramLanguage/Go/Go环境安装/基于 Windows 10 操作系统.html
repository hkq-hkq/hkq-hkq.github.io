# 安装Go安装包

双击下载好的安装包执行安装即可，建议将 Go 安装在 D 盘

# 创建工作空间

Go 代码固定存放在工作空间内，工作空间结构如下所示，创建对应的文件目录结构，其中bin 目录用于存放编译后的二进制文件，pkg目录用于存放编译后的包文件 ，src目录用于存放源文件：

    ├── pkg
    └── src

# 删除用户变量

当用户变量与系统变量发生冲突时，用户变量的优先级大于系统变量，则我们需要删除 Go 在安装时初步设置的用户变量，防止这些变量与我们后续设置的系统变量发生冲突，需要删除的用户变量如下：

  • GOPATH
  • Path 中的 %USERPROFILE%\go\bin 参数

# 编写并运行Go程序

在 src 文件夹下创建 hello.go 文件，并写入下述所示的程序代码：

//作用：在输出窗口上打印 Hello, World
package main

import "fmt"

func main() {
    fmt.Println("Hello, World")
}

执行下述 cmd 指令，编译并执行 Go 程序：

go run hello.go

若出现下述结果，则表示 Go 环境已完整安装！

Hello, World
